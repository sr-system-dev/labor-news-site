# Weekly SR by iand システム説明

## このシステムは何をするもの？

**労務・人事に関するニュースを自動で集めて、見やすいWebページにまとめるシステムです。**

毎週月曜日の朝9時に、過去1週間分のニュースを自動収集し、許可されたユーザーのみがアクセスできるWebページとして公開します。

## よくある質問

**Q: 費用はかかりますか？**
A: AIサマリー機能のため Claude API 利用料が月額約30円程度かかります。それ以外は無料です。

**Q: 誰かが操作する必要はありますか？**
A: ありません。毎週自動で更新されます。

**Q: スマートフォンでも見られますか？**
A: はい、スマホ・タブレット・PCすべてで見やすく表示されます。

**Q: 手動で更新することはできますか？**
A: はい、管理画面から「Run workflow」ボタンを押すだけで即時更新できます。

**Q: 特定の日のニュースがない場合がありますが？**
A: 各サイトが配信するRSSフィード（ニュース配信データ）には保持期間があり、古いニュースは自動的に消えます。また、その日にニュースが配信されなかった場合も空になります。

**Q: 一部のサイトからしか収集できていない場合は？**
A: サイト側の一時的な問題や、RSSフィードのURL変更などが考えられます。定期的に確認し、必要に応じてURLを更新します。

**Q: 誰でも閲覧できますか？**
A: いいえ。Cloudflare Access で認証されたユーザーのみ閲覧可能です。アクセスには管理者による招待が必要です。
